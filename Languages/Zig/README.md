# Zig

[← Back to Languages](../README.md)

**Why in DevOps and security:** **Zig** is a systems language (C alternative) with no hidden allocations and first-class C interop. Used in **Bun**, some systems tooling, and emerging security/low-level projects. Relevant when you encounter Zig-built binaries or contribute to infra written in Zig.
