# R

[← Back to Languages](../README.md)

**Why in security:** **R** is used in **forensics**, **threat intelligence**, and **security analytics**: statistics, visualization, and data pipelines. SOC and research teams use it for log analysis, IOC correlation, and reporting. Not for writing exploits or malware, but for understanding and communicating security data.

**Relevance:** When you need to analyze large datasets, build dashboards, or run statistical checks on security events, R (and its ecosystem) often appears. A bit of R helps with custom analytics and reading others’ scripts.
