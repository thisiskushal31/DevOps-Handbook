## Basic understanding (from official docs)

**Elixir** is a dynamic, functional language for building scalable and maintainable applications. It runs on the **BEAM** (Erlang VM), giving access to Erlang/OTP’s concurrency, fault tolerance, and distribution. The official docs are split into six applications: **Elixir** (standard library), **Mix** (build tool), **IEx** (interactive shell), **ExUnit** (testing), **Logger**, and **EEx** (templating). The project recommends the Getting Started guide and the Learning page for books and courses. In DevOps, Elixir shows up in **Phoenix** web apps and APIs, **Nerves** for embedded/IoT, and tooling that benefits from BEAM’s concurrency and reliability. Versioned docs are available on HexDocs (e.g. Elixir, Mix, IEx) for each release.
