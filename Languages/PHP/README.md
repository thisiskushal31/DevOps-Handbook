## Basic understanding (from official docs)

**PHP** (recursive acronym for *PHP: Hypertext Preprocessor*) is a widely used, open source, general-purpose scripting language especially suited for web development and embeddable in HTML. The PHP Manual is available in many languages; it covers getting started, language reference (basic syntax, types, variables, operators, control structures, functions, OOP, namespaces, errors), and extensions. In DevOps, PHP is the language of many web applications (WordPress, Laravel, Drupal) that you deploy and run; you work with PHP-FPM, Composer, and runtime/config rather than writing PHP day to day. The manual is the canonical reference for syntax, behavior, and deployment-related details.
