# Ruby

[← Back to Languages](../README.md)

**Why in DevOps:** Chef is written in Ruby; cookbooks and recipes are Ruby DSLs. Vagrant (Vagrantfile), Capistrano, and some legacy automation use Ruby. Less dominant than Python today but still present in many shops.

**Chef and this section:** This folder covers **Ruby as a language** so you can read and write Chef recipes, Vagrantfiles, and automation scripts. Chef *tool* coverage (workflow, cookbooks structure, Chef Infra, compliance, pipelines) lives under **IAC**: [IAC/Chef](../../IAC/Chef/README.md). Use Ruby here for the language; use IAC/Chef for Chef implementation and use cases.
