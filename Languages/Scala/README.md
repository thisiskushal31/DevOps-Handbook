# Scala

[← Back to Languages](../README.md)

**Why in DevOps:** Scala runs on the JVM; used in **Apache Spark**, **Kafka** (and Kafka tooling), and many data pipelines and streaming jobs. **sbt** (Scala build tool) is common; some services and APIs are written in Scala. You’ll encounter it in data engineering and JVM-heavy stacks.
