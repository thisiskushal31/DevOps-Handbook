# VBA and VBScript

[← Back to Languages](../README.md)

**Why in security:** **VBA** (Visual Basic for Applications) powers **macro malware** in Microsoft Office (Word, Excel). **VBScript** (.vbs) is used in legacy Windows scripting, droppers, and malware. Both are common in phishing and initial access; analysts and defenders need to read and deobfuscate them.

**Relevance:** Malicious documents and script-based payloads remain widespread. Understanding VBA/VBScript syntax, Office object model, and common obfuscation patterns supports detection, sandbox analysis, and incident response.
