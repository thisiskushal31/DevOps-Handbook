# Solidity

[← Back to Languages](../README.md)

**Why in DevOps and security:** **Solidity** is the main language for **Ethereum smart contracts**. Web3 DevOps involves building, testing, and deploying contracts; security focuses on vulnerabilities (reentrancy, overflow, access control). Understanding Solidity is essential for Web3 infra and smart-contract security.
